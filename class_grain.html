Detailed Description

This class creates and manages the Ready, Attack, Sustain, and Release states for individual grains. A grain is a very small segment of an audio segment, allowing for more granular synthesis and management of the waves that are outputted through VibroSonics hardware.

More information about grains and Attack Sustain Decay Release curves can be found at:

https://en.wikipedia.org/wiki/Granular_synthesis

https://en.wikipedia.org/wiki/Envelope_(music)
